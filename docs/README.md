# Atlas Documentation

Atlas is a Pan-African maps platform built in Rust — tile serving, geocoding, routing, and place search optimized for African road networks, addressing patterns, and languages.

  - [Search](api-reference/search.md) — POI and place search

  - [Telemetry](api-reference/telemetry.md) — Trip tracking and ETA learning

- [Self-Hosting](self-hosting.md) — Run Atlas on your own infrastructure
